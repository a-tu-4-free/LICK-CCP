## 這是什麼？

一個以國民黨親中統戰行為、議場暴力、土地豪宅、犯罪民代、立法維權爭議為題材的**純前端嘲諷器官證明書生成器**。

按一個按鈕，系統自動產生一份「統戰器官捐贈證明書」，內容包含器官中介、估價金額、兩面人語錄，以及北京評語。

**沒有後端。沒有資料庫。沒有追蹤。只有梗。**

## 玩法說明

| 按鈕 | 功能 |
|------|------|
| 🫀 隨機摘取 | 切換為隨機模式並生成器官證明，估價較低 |
| 🔴 屠城模式 | 切換為屠城模式並生成，估價更高、被抓包指數暴漲 |
| 🎭 屠城模式 ON/OFF | 開啟後器官證明同時顯示「台灣說法」與「北京說法」兩套話 |
| 🧬 器官升級 | 升級（Lv.1–10），等級越高估價越高、彩蛋越多 |
| 📖 統戰名冊 | 查看你已解鎖的統戰角色 |
| 📦 留下證據 | 下載截圖 |

### 器官證明互動
- **✏️ 編輯**：點擊後可直接修改器官證明內文，再截圖分享
- **🔄 再摘一次**：不滿意？馬上重抽
- **📤 分享**：一鍵分享到 FB、Twitter、Threads、LINE

### 隱藏系統
- **被抓包指數**：每次生成都會上升，突破 100 後有機率觸發 Glitch 語句
- **偽裝成功率**：被抓包越多，偽裝成功率越低——台媒狗仔抓包機率同步上升
- **成就系統**：完成特定行為解鎖成就徽章（點「▼ 成就」展開查看）
- **今日任務**：每個 Session 隨機指派一個任務，完成解鎖成就
- **統戰名冊**：多位統戰角色，每次生成器官證明隨機解鎖，點擊查看專屬語錄
- **台媒狗仔抓包**：隨時可能觸發，全螢幕蓋住畫面 3 秒

## 技術架構

```
lick-ccp/
├── index.html        # 主頁面、UI、Modal、社群分享邏輯
├── script.js         # 遊戲核心引擎（ES Module）
├── generator.js      # 器官估價、器官中介名生成器
├── ui.js             # 器官證明動畫、Glitch 特效、評語渲染
├── style.css         # 全站樣式（中共官方紅金風）
├── memes.json        # 所有梗文字資料庫
├── characters.json   # 統戰角色資料（emoji + 專屬語錄）
└── 啟動伺服器.bat    # 本地端一鍵啟動（Windows）
```

**使用的外部資源：**
- [html2canvas](https://html2canvas.hertzen.com/)（截圖功能）

## 本地端運行

1. 下載或 Clone 此 repo
2. 雙擊 `啟動伺服器.bat`
3. 瀏覽器自動開啟 `http://localhost:8080`

> 需要 Python 3 或 Node.js，BAT 會自動偵測。
